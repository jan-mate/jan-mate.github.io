Keys missing on most layouts

Most layouts fail to include many useful keys, because they wrongly just permutate the standard 104 keys on most layouts.

Here is a list of keys/functions I think you should consider adding to your keyboard layout.

Text navigation

  • Mark line
  • Duplicate Line
  • Mark word

But doesn't Vim already do these? Yes, but Vim doesn't work as many places as this does.

I've noticed some layouts don't use pgup, pgdn, home and end. Those are some of the most underrated keys (i guess that regular layouts have them in horrible position, so ppl don't realize how useful they are)

Navigation

  • Switch tabs (ctrl+pgup/pgdn)
    S-tier shortcut, very underrated. Works in most software
